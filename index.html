CARGO: A Framework for Confidence-Aware Routing of Large Language Models
A. Barrak, Y. Fourati, M. Olchawa, E. Ksontini, K. Zoghlami  ·  Oakland University & Mediterranean Institute of Technology
CARGO is a lightweight, confidence-aware framework for dynamic LLM selection. It employs a single embedding-based regressor trained on LLM-judged pairwise comparisons to predict model performance, with an optional binary classifier invoked when predictions are uncertain. Evaluated on GPT-4o, Claude 3.5 Sonnet, DeepSeek V3, and Perplexity Sonar achieving 76.4% top-1 accuracy and 72–89% win rates against individual models.

🎓 Bachelor's Capstone Thesis  ·  Mediterranean Institute of Technology, 2024–2025
Ensemble Model Selection for LLMs: Does Picking the Best Model Improve Response Quality?
Yosr Fourati  ·  Supervised by Dr. Dorra Louati (MedTech) & Dr. Amine Barrak (Oakland University)
Introduced a dynamic routing framework that routes prompts to the best-suited LLM among four state-of-the-art models (DeepSeek V3, Claude-3.5 Sonnet, GPT-4o, Perplexity Sonar) across five task categories. The framework achieves win rates of 65–90% over individual models, with a top-1 accuracy of 74.6% using DeBERTaV3Small. Category-specific routing with Random Forest reaches 84.9% accuracy on coding tasks.
